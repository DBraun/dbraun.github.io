PyTorch-TOP

C++ TouchDesigner plugin for real-time style transfer with PyTorch.

My public Github repo for this project has some CPU-side overhead, but in a private version, I’ve eliminated it. Instead I use CUDA to keep all data on the GPU, avoiding slow CPU-to-GPU or GPU-to-CPU data transfer. I’ve done the same CUDA project with the TensorFlow C++ API too.
